# My Demo Collection 📁

这是一个用于存储**个人开发 Demo 和小项目**的仓库，每个项目独立存放在自己的文件夹中，并附带专属说明文档。

---

### 🌟 特点
- **独立管理**：每个项目存放在单独文件夹内（如 `/vue3-snake`）
- **自述清晰**：每个项目均有自己的 `README.md`，包含运行指南和效果说明
- **技术多样**：包含前端框架、工具脚本、算法实现等不同类型的实践
